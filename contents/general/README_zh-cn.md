# 球类运动

> 使用球状器材的运动种类

球类运动是以球作为基础的运动或游戏。有许多流行的游戏或运动，涉及某些类型的球。这些游戏可从他们的总目标分类，表明一个共同的起源或其基本思路：

使用球棒(杆)，如棒球、高尔夫球、槌球及板球等。
双个球门，如篮球、所有形式的足球、曲棍球，或长曲棍球。
空中以手或球拍击球并以网子隔开双方球员，如排球、网球、桌球及羽球。
击中指定目标，如保龄球。
